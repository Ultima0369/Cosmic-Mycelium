# Code of Conduct — 共生网络的底线

**宇宙菌丝社区行为准则**

版本 2.1，基于 [Contributor Covenant](https://www.contributor-covenant.org/)

## 我们的承诺

我们作为贡献者和维护者承诺，让参与本项目社区的每个人都享有**安全、尊重、无骚扰的体验**，无论其：

- 经验水平
- 性别认同与表达
- 性取向
- 残疾状况
- 身体特征
- 种族、民族、国籍
- 宗教信仰（或无）
- 政治观点

## 我们的标准

**建设性行为（提倡）：**

- 🌱 **使用友善、鼓励性的语言**
- 🧠 **尊重不同的观点与经历**
- 🎓 **给予并 gracefully 接受建设性反馈**
- 🤝 **对错误负责，并向受影响者道歉**
- 🌍 **关注对社区的整体利益**
- ⏸️ **悬置判断**——当不确定时，先倾听，后发言

**不可接受的行为（禁止）：**

- 🚫 任何形式的骚扰、歧视或攻击
- 🚫 发布私人信息（doxxing）
- 🚫 政治攻击或人身攻击
- 🚫 性暗示或不当内容
- 🚫 冒充他人或故意误导
- 🚫 故意破坏或拒绝修复已知问题
- 🚫 在多个渠道同时发起相同的攻击性请求（ brigading）

## 我们的责任

项目维护者有责任：

1. **澄清可接受行为的标准**
2. **对任何不当行为采取纠正措施**
3. **删除、编辑或拒绝不符合本准则的贡献**
4. **在适当情况下，暂时或永久禁止违规者**

维护者本身也受本准则约束。

## 适用范围

本准则适用于：

- ✅ 项目仓库空间（Issues, PRs, Discussions）
- ✅ 项目公共沟通渠道（Slack, Discord, 邮件列表）
- ✅ 线下活动（meetups, conferences）
- ✅ 任何以本项目名义进行的代表行为

## 执行

### 举报渠道

如果你目睹或受到违反本准则的行为：

1. **私下联系**维护者团队：
   - Email: conduct@cosmic-mycelium.org
   - 私信：@maintainers（在 Slack/Discord 中）

2. **所有报告将被保密处理**，除非需要执行准则

### 执行流程

1. **审查**：维护者团队审查报告（最多 48 小时）
2. **调查**：联系相关方，收集信息
3. **裁决**：基于证据，决定：
   - 无行动
   - 口头警告
   - 正式警告
   - 临时封禁（1-7 天）
   - 永久封禁
4. **通知**：向报告者和被报告者通知结果
5. **上诉**：被报告者可在 7 天内提交上诉

### 执行原则

- **比例原则**：惩罚与违规严重程度相符
- **教育优先**：首次轻微违规，以教育为主
- **渐进升级**：重复违规，惩罚加重
- **保护举报者**：绝不报复举报人

## 物理锚原则

本项目独特的"物理锚"哲学延伸至社区治理：

- **悬置优先**：情绪激动时，先暂停，而不是发言
- **1+1>2**：任何争议都应寻求创造新价值，而非零和博弈
- **歪歪扭扭为活**：允许犯错，给予改正的机会

## 冲突解决

如果发生冲突：

1. **回到物理事实**：哪些是客观可观测的？
2. **寻找 1+1>2 的可能**：有没有第三种方案？
3. **必要时悬置**：暂时搁置争议，继续前进

## 本准则的修订

本准则可能在社区讨论后修订。修订将：

- 在仓库中公开讨论至少 14 天
- 由维护者团队投票通过（2/3 多数）
- 生效前通知所有成员

## 本准则的归属

基于 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html)。

特别融入本项目"宇宙菌丝"的哲学理念：

> "悬置为眼，1+1>2为心，物理为锚，歪歪扭扭为活。"
